# Frontend Screenshots

Add your Master Auth frontend screenshots to this directory to display in the main README.md

## Instructions

1. **Take screenshots** of each major page:
   - `home-page.png` - Home page with mission briefing
   - `register-page.png` - Registration form
   - `login-page.png` - Login form
   - `dashboard.png` - Dashboard showing tokens and data

2. **Image Requirements:**
   - Format: PNG or JPG
   - Size: Approximately 1200x800 pixels
   - Include any relevant UI elements
   - Ensure no sensitive data is visible

3. **Recommended Tools:**
   - Snagit
   - Lightshot
   - Built-in Windows Screenshot (Windows + Shift + S)
   - VS Code Screenshot extension

## Screenshot Locations

Each screenshot should capture:

### home-page.png
- Terminal window with ASCII art
- Challenge briefing section
- Objectives list
- Navigation links (Login/Register)

### register-page.png
- Registration form with:
  - Username input field
  - Email input field
  - Password input field
  - Register button
  - Login link

### login-page.png
- Login form with:
  - Username/Email input field
  - Password input field
  - Login button
  - Register link

### dashboard.png
- User information display
- JWT Token section (tokenized view)
- Dashboard data
- Refresh token button
- Logout button

After adding the screenshots, the README will automatically display them in the Screenshots section.
